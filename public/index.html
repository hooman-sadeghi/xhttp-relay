چرا XHTTP Relay؟

سرعت و پایداری بالا

فوروارد مستقیم ترافیک XHTTP با کمترین تأخیر

امنیت کامل

پشتیبانی از TLS + هدر هوشمند x-host

بدون محدودیت

برخلاف Netlify، هیچ محدودیتی در ترافیک و تعداد درخواست ندارید

نحوه کارکرد

۱

دامنه Coolify

دامنه‌ای که Coolify به شما می‌دهد یا دامنه شخصی خودتان

۲

ارسال هدر x-host

کلاینت شما هدر x-host را با آدرس سرور اصلی ارسال می‌کند

۳

فوروارد هوشمند

ریلی ترافیک را به سرور Xray شما (مثلاً پورت ۴۴۴) منتقل می‌کند

{
  "outbounds": [
    {
      "protocol": "vless",
      "settings": { ... },
      "streamSettings": {
        "network": "xhttp",
        "security": "tls",
        "xhttpSettings": {
          "path": "/IR_NETLIFY",
          "host": "your-relay-domain.com"
        }
      }
    }
  ]
}
